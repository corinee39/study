## 5.5 문자열 타입

### 문자열 비교 

- [소스](EqualsExample.java)
- 문자열 리터럴로 생성 VS new 연산자로 생성
  - 동일한 문자열 리터럴로 객체를 생성 -> `참조가 같음`
  - new 연산자로 새로운 객체 생성 -> `참조가 다름`
- 동일한 객체든 다른 객체든 상관없이 내부 문자열만 비교 => `equals()`

### 문자 추출 - `charAt()`
- [소스](CharAtExample.java)

### 문자열 관련 메소드
- `length()` - 문자열에서 문자의 개수 얻기
- `replace()` - 특정 문자열을 다른 문자열로 대체
- `substring()` - 특정 위치의 문자열을 잘라내 가져오기
- `indexOf()` - 특정 문자열의 위치 찾기 / 인덱스를 리턴
- `contains()` - 문자열이 단순히 포함되어 있는지 여부만 조사 - [소스](IndexOfContainsExample.java)
- `split()` - 구분자를 사용하여 문자열을 따로 분리
